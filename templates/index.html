About
ClipperZipper is a web scraping tool for retrieving chat logs from OverRustleLogs, a website which logs the chat messages for several hundred Twitch channels.

On OverRustleLogs, you can only view the monthly logs of a single user at once. But with ClipperZipper, you can retrieve logs from a more specified timeframe, and you can view the logs of multiple specified users at once. This makes viewing conversations in chat easy.

Help
To use ClipperZipper, start by providing a valid channel name. The list of available channels can be viewed here.

Then, provide a list of users whose logs you wish to retrieve, separating each user with a comma and a space (e.g., <username1>, <username2>).

Logs are retrieved from the current time (UTC) back to the past X months/weeks/days/hours.

The "mentions only" setting means that the logs should only consist of messages where the users mention each other. This is standard for viewing conversations. Otherwise, all of those users' logs from the specified time will be retrieved.
